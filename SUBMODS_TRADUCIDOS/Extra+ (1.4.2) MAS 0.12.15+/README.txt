EXTRA+ SUBMOD FOR MONIKA AFTER STORY
====================================

Thank you for downloading the Extra+ Submod!

This expansion adds new dating locations, minigames, and interactions with Monika.


============================================================================== 
INSTALLATION INSTRUCTIONS
==============================================================================

IMPORTANT: Please install this submod ONLY when Monika is asleep or the game is closed.

1. Locate your "Doki Doki Literature Club" main folder.
   (This is the folder containing "DDLC.exe".)

2. Open the "game" folder inside your DDLC directory.

3. Open the ZIP file you just downloaded.

4. Drag and drop the contents of the ZIP file (the "Submods" and "mod_assets" folders)
   directly into your "game" folder.

   - If Windows asks to "Merge folders", click "Yes".
   - If Windows asks to "Replace files", click "Yes" (this updates old versions).

5. Launch Monika After Story.

6. Verify the installation:
   - Look for the "Extra+" button in the main menu (on the left side).
   - If you see the button, congratulations! The submod is installed correctly and ready to use.
   - If the button is NOT visible, the installation was incorrect. Please review the steps above
     and ensure files are in the right location.


============================================================================== 
TROUBLESHOOTING
==============================================================================

- If you see a gray screen or errors about missing images:
  You likely placed the files in the wrong location. Ensure that the "ExtraPlus"
  folder is inside "game/Submods/" and not just sitting in the "game" folder directly.

- Compatibility:
  This submod requires the latest version of Monika After Story.
